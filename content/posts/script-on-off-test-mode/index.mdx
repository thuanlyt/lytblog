Tạo file .bat bật/tắt test mode sẽ cực kì tiện và cũng không quá xa lạ đối với những anh em thường xuyên bật tắt... để làm gì thì cũng dễ hiểu chứ nhỉ haha.

Thông thường để bật **test mode** chúng ta sẽ phải chạy lệnh cmd như sau
`bcdedit -set TESTSIGNING ON`
và tắt thì cũng tương tự, chỉ cần thay giá trị **ON** thành **OFF** để tắt.
Nhưng cách này sẽ khá bất tiện đối với những ai thường xuyên bật tắt. Mình sẽ chỉ mẹo tạo file **.bat** ngay bên dưới nhé!

Đầu tiên tạo 1 file `text.txt` và đổi tên nó thành `ON-TEST-MODE.bat` (tùy các bạn miễn là đuôi `.bat` nha)
Sau đó mở lên bằng notepad, nhập code sau và save lại:

```bat title=ON-TEST-MODE.bat highlight=5-7,10
@echo off
color 0e
bcdedit -set TESTSIGNING ON
chcp 65001
CLS
echo TEST MODE: ĐÃ BẬT
pause
```

Tương tự ta sẽ tạo 1 file `OFF-TEST-MODE.bat` và nhập mã sau:

```bat title=OFF-TEST-MODE.bat highlight=5-7,10
@echo off
color 0e
bcdedit -set TESTSIGNING ON
chcp 65001
CLS
echo TEST MODE: ĐÃ BẬT
pause
```

Save lại, và xong rồi. Từ giờ chỉ cần double click vào file **.bat** là oke r nha!
